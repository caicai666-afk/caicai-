本项目是一个利用通用智能体（如 Manus、coze空间、minimax agent等）自动生成结构化的海外行业新闻日报，并通过企业微信群机器人进行推送的自动化解决方案。

当前示例场景：南非汽车金融行业。每日自动抓取最新资讯，按政策市场、比亚迪动态、竞品动态三类归纳，生成包含数据、影响和来源链接的日报，并推送至企业微信群。

✨ 功能特点  

自动化日报生成：定时触发，无需人工干预。

结构化输出：严格按照预设分类（行业政策、比亚迪动态、竞品动态）组织内容，包含具体事件、数据和影响分析。

多源信息整合：通过联网搜索或新闻 API 获取可靠信源的最新新闻。

企业微信直推：生成 markdown 格式的日报，直接发送到企业微信群，支持图标、加粗等排版。

高度可定制：修改提示词即可适配不同行业、地区或语言。

🛠️ 技术栈

- Manus（或其他兼容的通用智能体）

- 企业微信机器人 Webhook
企业微信群机器人 Webhook 地址（替换 key 部分）：WECHAT_WEBHOOK_URL=https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=your_webhook_key

💬 核心提示词：  

生成一份结构清晰、内容客观的新闻日报，字数限制在400-800个汉字。从可靠新闻渠道获取南非本月最近几天的汽车金融行业新闻，按南非行业政策与市场变化＆汽车关税、南非比亚迪相关动态、南非竞品与行业相关动态三类归纳，每类提供2-4条具体新闻，每条新闻需包含事件、数据、影响等具象化细节。大类之间空一行，使得格式更清楚。使用标题、项目符号和小图标等（如📊、🚗、⚔等图标）增强可读性，每类新闻前加粗标题，新闻条目用列表形式，结尾添加总结段概括本月关键趋势，以markdown语法标注新闻具体来源链接。文字抬头为“南非汽车金融行业新闻日报”。 最后，将文字整理为markdowm格式，并且通过python脚本推送到企业微信群机器人（ "msgtype": "markdown",）：https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=your_webhook_key

如果一切正常，你将在企业微信群中收到一条 markdown 格式的新闻日报。你可以按需调整行业、地区、分类和格式要求，修改提示词。


⚙️ 工作原理  

获取新闻：通用智能体调用大模型 API，并启用联网搜索功能（如 OpenAI 的 gpt-4-with-browsing 或通过外部搜索API将结果作为上下文）。

生成日报：大模型根据提示词，将搜索结果整理成结构化的 markdown 文本，并附上来源链接。

推送微信：使用 requests 库将 markdown 内容发送到企业微信群机器人 Webhook。

🔧 自定义指南  

更换行业/地区：修改提示词中的“南非汽车金融”为任何你关心的领域，例如“巴西农业机械”、“东南亚新能源”等。

调整分类：在提示词中修改分类名称和数量，模型会自动适应。

增加字数或详细程度：修改提示词中的字数限制和条目数量。

新闻来源控制：如果你有特定的新闻源偏好，可以在提示词中明确要求“优先使用 Reuters、Bloomberg 等”。

⚠️ 注意事项  

API 费用：每次运行会消耗大模型 API 的 token，请留意费用。启用联网搜索功能可能额外计费。

新闻时效性：确保模型具备联网能力，否则只能依赖训练数据中的旧知识。建议使用支持实时搜索的模型，或通过外部搜索工具补充最新信息。

企业微信机器人限制：每个机器人每分钟最多发送 20 条消息，markdown 内容需符合企业微信语法。

内容可靠性：尽管提示词要求“可靠新闻渠道”，但模型可能产生幻觉，建议人工抽查或限制来源域名。
